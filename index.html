HTML

  • The body element represents the visible content shown to the user
  • Information stored in the head element improves SEO and what language to expect
  • Only use one H1 element per page in descending order of importance

CSS

  • CSS is used for stylizing the page
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Using classes makes it easy to style different sections of a webpage in the same way
  • For a stylesheet to apply to a webpage you must link it in the html file within the head

Git

  • Git is very important to keep iterations of a project
  • git status checks what branch you are currently in
  • git checkout -b "" makes a new branch and switches to it

JavaScript

  • JavaScript is important to make a website functional and accesable to the user
